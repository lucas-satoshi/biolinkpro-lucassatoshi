# BioLink Pro - Sua Vitrine Digital

[cite_start]Um projeto de cartão de visitas digital desenvolvido para reunir canais de atendimento, redes sociais e portfólio em um único link[cite: 5, 18]. [cite_start]Este projeto foi construído como parte do desafio "Fase 1 - Projeto 2" [cite: 4] [cite_start]promovido pela SOFTEX, NUCLEO CAMPINAS e trendsIT[cite: 1, 2, 3].

[cite_start]O foco central desta aplicação é a construção de uma interface leve, rápida e otimizada para dispositivos móveis, utilizando fundamentos sólidos de Web Design, estruturação semântica e estilização pura, sem a dependência de plataformas genéricas ou frameworks[cite: 7, 12, 119].

## 🚀 Tecnologias e Conceitos Aplicados

[cite_start]Este projeto foi construído estritamente com tecnologias front-end base[cite: 122]:

* [cite_start]**HTML5 Semântico:** Utilização correta de tags como `<header>`, `<main>`, `<nav>` e `<footer>` para garantir acessibilidade e boa interpretação por motores de busca[cite: 41, 73, 116].
* [cite_start]**CSS3 Vanilla:** Toda a estilização, paleta de cores e tipografia foram feitas do zero em um arquivo externo `.css`, consolidando o domínio do Box Model (margin, padding, border) e unidades relativas[cite: 75, 82, 116].
* [cite_start]**Flexbox:** Alinhamento e distribuição harmônica de todos os elementos na tela, tanto no eixo vertical quanto no horizontal, garantindo uma interface equilibrada[cite: 44, 45, 117].
* [cite_start]**Mobile-First e Responsividade Nativa:** Layout desenhado prioritariamente para telas de celulares, ajustando-se automaticamente a qualquer resolução sem quebras ou barras de rolagem horizontais[cite: 66, 69, 113].
* [cite_start]**Tipografia:** Integração com o Google Fonts para aplicação de fontes modernas e legíveis[cite: 43].

## ⚙️ Funcionalidades

* [cite_start]Perfil centralizado contendo foto com recorte circular, nome e breve biografia profissional[cite: 35, 110].
* [cite_start]Lista de links estratégicos organizados em botões com estilização consistente[cite: 36, 110].
* [cite_start]Interatividade visual com efeitos de transição (`hover` e escala) ao interagir com os botões[cite: 36, 111].
* [cite_start]Abertura segura de links externos utilizando `target="_blank"`[cite: 154].
* [cite_start]Botão em destaque com cores personalizadas para redirecionamento direto ao WhatsApp[cite: 36].
