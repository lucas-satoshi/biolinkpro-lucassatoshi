# BioLink Pro - Sua Vitrine Digital

Um projeto de cartão de visitas digital desenvolvido para reunir canais de atendimento, redes sociais e portfólio em um único link. Este projeto foi construído como parte do desafio "Fase 1 - Projeto 2" promovido pela SOFTEX, NUCLEO CAMPINAS e trendsIT.

O foco central desta aplicação é a construção de uma interface leve, rápida e otimizada para dispositivos móveis, utilizando fundamentos sólidos de Web Design, estruturação semântica e estilização pura, sem a dependência de plataformas genéricas ou frameworks.

## 🚀 Tecnologias e Conceitos Aplicados

Este projeto foi construído estritamente com tecnologias front-end base:

* **HTML5 Semântico:** Utilização correta de tags como `<header>`, `<main>`, `<nav>` e `<footer>` para garantir acessibilidade e boa interpretação por motores de busca.
* **CSS3 Vanilla:** Toda a estilização, paleta de cores e tipografia foram feitas do zero em um arquivo externo `.css`, consolidando o domínio do Box Model (margin, padding, border) e unidades relativas.
* **Flexbox:** Alinhamento e distribuição harmônica de todos os elementos na tela, tanto no eixo vertical quanto no horizontal, garantindo uma interface equilibrada.
* **Mobile-First e Responsividade Nativa:** Layout desenhado prioritariamente para telas de celulares, ajustando-se automaticamente a qualquer resolução sem quebras ou barras de rolagem horizontais.
* **Tipografia:** Integração com o Google Fonts para aplicação de fontes modernas e legíveis.

## ⚙️ Funcionalidades

* Perfil centralizado contendo foto com recorte circular, nome e breve biografia profissional.
* Lista de links estratégicos organizados em botões com estilização consistente.
* Interatividade visual com efeitos de transição (`hover` e escala) ao interagir com os botões.
* Abertura segura de links externos utilizando `target="_blank"`.
* Botão em destaque com cores personalizadas para redirecionamento direto ao WhatsApp.
